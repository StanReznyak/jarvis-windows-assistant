# Как устроен JARVIS

## Запуск

`main.py` создаёт приложение из `jarvis/app/main_window.py`. Основное окно собирает отдельные панели через mixin-классы: голос, напоминания, настройки, диагностику, действия и служебные окна.

## Локальные команды

`command_core.py` отвечает за нормализацию фраз, распознавание намерений и обработку вариантов произношения. Команды запуска программ, сайтов, папок, заметок и напоминаний выполняются локально и не требуют языковой модели.

## Голос

Распознавание речи по умолчанию работает через Vosk. Код также содержит дополнительные режимы для SpeechKit, faster-whisper и whisper.cpp, но они не обязательны для обычного запуска.

Озвучка отделена от распознавания речи. Во время ответа микрофон временно игнорируется, чтобы JARVIS не принимал собственный голос за новую команду.

## Ответы через модели

`jarvis/model_chat.py` выбирает локальный или облачный провайдер:

- локальный режим обращается к Ollama на `127.0.0.1:11434`;
- облачный режим используется только при наличии пользовательского ключа;
- при недоступности модели обычные локальные команды продолжают работать.

## Пользовательские данные

Рабочие данные не лежат в исходниках. В портативном режиме они сохраняются в `data` рядом с приложением. В режиме обычной установки можно использовать `%APPDATA%\JARVIS`.

Основные файлы:

- `config.json` — настройки;
- `notes.txt` — заметки;
- `reminders.json` — напоминания;
- `memory_profile.json` — локальная память;
- `ui_session.json` — состояние интерфейса;
- `system_events.log` — журнал событий.

## Сборка

`build_exe.bat` запускает проверку, очищает старую сборку и собирает папку `dist\JARVIS` через PyInstaller.

`build_installer.bat` сначала вызывает сборку EXE, затем создаёт установщик через Inno Setup.

## Проверка перед релизом

`build_check.py` проверяет:

- единую версию во всех основных файлах;
- синтаксис Python-кода;
- безопасные настройки запуска;
- одноразовую миграцию старого конфига;
- кодировку и переносы строк BAT-файлов;
- отсутствие временных файлов и случайно добавленных ключей.
