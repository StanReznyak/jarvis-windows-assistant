Папка для локального установщика Ollama.

Чтобы добавить установщик Windows в свою сборку, положите файл сюда:

tools\ollama\OllamaSetup.exe

SETUP_LOCAL_MODEL_WINDOWS.bat сначала проверит этот файл. Если его нет, скрипт попробует установить Ollama через winget.

Сама языковая модель в проект не входит. Обычно она загружается командой:
ollama pull llama3.2:1b
