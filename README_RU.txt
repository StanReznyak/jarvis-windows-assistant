JARVIS v8.4

Обычный запуск:
1. Распакуйте архив в отдельную папку.
2. Откройте папку JARVIS.
3. Запустите START_JARVIS.bat.

Полезные файлы:
- run.bat — запуск из исходников;
- build_exe.bat — сборка EXE;
- build_installer.bat — сборка установщика;
- python build_check.py — проверка проекта;
- python -m unittest discover -s tests -v — короткие автотесты.

Если окно BAT сразу закрывается, откройте обычную CMD-консоль и выполните:

cd /d C:\JARVIS_test\JARVIS
START_JARVIS.bat

Для голосового ввода нужна распакованная модель Vosk в папке model.
В полной Windows-сборке модель может быть уже добавлена.

Окна первичной настройки и восстановления сессии по умолчанию отключены.
Старый config.json обновляется автоматически один раз.
